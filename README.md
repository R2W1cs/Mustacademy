**MUSTACADEMY** is a premium, AI-powered educational platform designed to guide computer science students through their academic journey with industry-grade roadmaps, real-time AI tutoring, and personalized career architecture.

---

## ✨ Key Features

### 🚀 Interactive Roadmaps
Structured learning paths that adapt to your university curriculum, ensuring you master core CS concepts while staying aligned with your studies.

### 🤖 AI Pilot
Your personal 24/7 CS tutor. Integrated into every topic, the AI Pilot provides explanations, debugs code, and offers deep-dives into complex algorithms.

### 🏗️ Career Architect
Generate professional trajectories based on your current year, semester, and dream job. The Career Architect bridges the gap between classroom theory and industry requirements.

### 📊 Progress Analytics
Track your mastery across different domains. Our dashboard provides granular insights into your learning velocity, "Brain Power," and academic focus.

### 🌓 Theme-Aware Interface
A sleek, premium UI that supports both **Celestial (Dark)** and **Aura (Light)** modes, optimized for long study sessions.

## 🛠️ Technology Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend** | React 18, Vite, Tailwind CSS, Framer Motion |
| **Backend** | Node.js, Express, Multer |
| **Database** | PostgreSQL (Hosted on Neon.tech) |
| **AI Engine** | Groq / Llama 3 / Gemini (with local fallback) |
| **Deployment** | Vercel (Frontend), Render (Backend) |

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/R2W1cs/Mustacademy.git
cd cs-roadmap-platform
```

### 2. Configure Environment Variables
Create `.env` files in both `frontend` and `server` directories following the `.env.example` templates.

### 3. Install Dependencies & Start
**Backend:**
```bash
cd server
npm install
npm start
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```
